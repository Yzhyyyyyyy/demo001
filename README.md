点击下方链接即可**直接查看对应游戏的抽卡页面源码**：

## 🛠️ 技术栈
- **语言**: Kotlin
- **UI**: Jetpack Compose
- **工具**: Android Studio

---

## 🚀 如何运行？

1. 克隆项目到本地：
   ```bash
   git clone https://github.com/Yzhyyyyyyy/demo001.git
   ```
2. 使用 Android Studio 打开 `demo001` 文件夹。
3. 点击顶部的 ▶️ (Run) 按钮即可在模拟器或真机上运行。
